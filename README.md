# Metodos-estadisticos-Vciclo
Este repositorio ha sido creado como parte del curso Sistematización y Métodos Estadísticos.
Aquí se almacenarán los archivos, ejercicios, códigos y materiales desarrollados durante el ciclo académico.

El objetivo de este repositorio es:
- Organizar de manera sistemática las actividades del curso.
  
- Gestionar los diferentes trabajos realizados.
  
- Facilitar el acceso, revisión y actualización de los proyectos.
  
- Documentar los análisis y métodos estadísticos aplicados.
